Personal Projects

Rhythmic Ride

1st Place for Best Use of Mojio API at Dubhacks 2014. Uses the Moj.io connected car platform to adjust music based on driving factors, currently speed. (For example, a driver who is too slow might need faster-paced music to get him/her to drive faster, and similarly a speeding driver should be subjected to calmer music).

Merit Badge App

Boy Scout Merit Badge App (17,000+ downloads) provides simple user interface for Scouts to look up requirements for popular and required merit badges.

ConnectMore App

ConnectMore app (1,700+ downloads) is a variation of the Connect Four game in which you get more points if you win in multiple ways. If you get 5 in a row, you win in two ways, so you get 2! (2) wins. If you get 6 in a row, you win in three ways, so you get 3! (6) wins, and so on.

Distress Detector

Integrates Microsoft band functionality with an Android App to sense whether the user is distressed when they are in potentially dangerous situations. It monitors the user's heart rate to see whether the user is in distress. If so, the app will send a message to the user's emergency contacts.
